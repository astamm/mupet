# Changelog

## mupet 0.1.0

Implementation of custom ‘tidymodels’ metrics for multi-class prediction
models with a single negative class. Currently are implemented:

- macro-average sensitivity through
  [`macro_average_sensitivity()`](https://astamm.github.io/mupet/reference/macro_average_sensitivity.md);
- macro-average specificity through
  [`macro_average_specificity()`](https://astamm.github.io/mupet/reference/macro_average_specificity.md);
- a generalized weighted Youden index through
  [`weighted_youden_index()`](https://astamm.github.io/mupet/reference/weighted_youden_index.md).
